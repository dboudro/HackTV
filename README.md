##App Features


####Sublime Text Integration
 - report: streamer theme and plugins
 
####Github Integration
  - widget for: pushes and errors

 - ability to be a celebrity users (e.g: swift, etc)
 - streamer journals (prints out time coding, keystrokes, etc)
